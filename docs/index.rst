About jason
===========

jason is a set of lightweight helpers that makes it simpler to return JSON
http responses from a Django view.

Installation
============

Can be downloaded from `PyPi
<https://pypi.python.org/pypi/jason/>`_, or via a package manager::

    pip install jason
